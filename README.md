US High School Performance Dashboard

Project Overview

Built an interactive 4-page Power BI dashboard to analyze student performance across U.S. high schools. The dashboard explores academic outcomes by socioeconomic status (SES), school type, race, gender, grade level, and geographic location.

Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling
* Star Schema

Dashboard Features

* 4 interactive dashboard pages
* 10+ visualizations
* Cross-page filtering
* Drill-down analysis
* Decomposition Tree
* KPI Cards
* DAX Calculations

Key Insights

* Students in higher SES quartiles generally achieved stronger academic outcomes.
* Public schools showed slightly higher average total scores than private schools.
* Performance differences were identified across gender, race, and locale categories.
* Interactive filtering enabled detailed exploration of student achievement patterns.

Skills Demonstrated

* Dashboard Development
* Data Visualization
* Data Cleaning
* Data Modeling
* DAX
* Business Intelligence
* Analytical Reporting
